# tutorial
install the dependencies:
```sudo xbps-install -S i3 i3status firefox blueman bluez pasystray pipewire wireplumber feh NetworkManager xdg-desktop-portal setxkbmap mate-polkit font-iosevka alacritty zsh j4-dmenu-desktop playerctl slock scrot nemo```
it's also optional to build a boomer util (https://github.com/tsoding/boomer.git)

do the ```git clone --depth 1 https://github.com/wcupped/dotfiles.git```
then do the ```cd dotfiles```
then run `install.sh`
